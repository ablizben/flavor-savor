# Flavor Saver
An Instagram/Pinterest hybrid for discovering new food, restaraurants, and fellow foodies.

## Technologies
* Passport or Firebase
* Google Maps
* Lazy Loader
* React/JSX
* Node/Express
* MongoDB/Mongoose
* Heroku/Atlas
* Material UI
* MVC Paradigm
* Travis/ESLint
